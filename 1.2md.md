#今日认识并学习git心得：

##认识git

###一.**git**：

####1.实质是分布式版本控制系统

​           2.存在形式为本地安装的命令行

​           3.核心功能——可追踪代码版本、分支管理、本地提交、合并代码。

###二.**GitHub**

####1.基于git的代管平台

​          2.在线的web服务（网页）

​           3.核心功能是托管git仓库，提供远程协作（可用于小组合作时远程的代码交换及修改），可视化管理、团队权限管理等。

​           底层基于git实现版本管理。

##今日实操及GitHub所有的命令

###1.调出命令窗口（win＋r,输入cmd＋回车）

###2.配置全局用户名及邮箱（git config --global user.name "你的名字"  

​                                                     git config --global user.email "你的邮箱"  ）

###3.构建心得文件夹

###4.在目录初始化仓库（git init）

###5.添加当前目录所有变更文件到暂存区（git add .）

###6.暂存区文件提交到本地仓库（git commit -m "提交说明"）

###7.以及列出分支、仓库等
